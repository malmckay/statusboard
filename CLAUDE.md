# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

Prefer `just <recipe>` — see `Justfile` for all recipes.

```bash
# Install dependencies
npm install

# Worker — local dev (seeds D1 local DB, then starts dev server)
just dev          # or: pnpm dev

# Worker — deploy to production (runs migrations first via predeploy hook)
just deploy       # or: npm run deploy

# Worker — typecheck + dry-run deploy
just check        # or: npm run check

# Worker — generate Cloudflare Worker types
just typegen      # or: npm run cf-typegen

# Inkplate — compile Arduino sketch
just compile

# Inkplate — compile + flash to connected device
just flash

# Apply migrations to local D1 only
npm run seedLocalD1
```

No test framework is configured.

## Architecture

This repo has two components:

### Worker (root)

Minimal Cloudflare Workers project backed by Cloudflare D1 (SQLite).

**Entry point:** `src/index.ts` — fetch handler; routes `/daily-image` to PNG generation, `/` to D1 demo.

**Image generation:** `src/dailyImage.ts` — fetches weather + joke, renders a 1280×720 PNG via satori (SVG layout) + @resvg/resvg-wasm. Top 280px left blank for the device to draw the clock.

**Database:** `migrations/` — applied via Wrangler. D1 binding is `DB` in `wrangler.json` and `worker-configuration.d.ts` (auto-generated by `wrangler types` — do not edit manually).

**D1 binding name:** `"DB"` (not the database name `"d1-statusboard-database"`). Wrangler commands use `DB` as the identifier (e.g., `wrangler d1 migrations apply DB --local`).

### Inkplate sketch (`inkplate/`)

Arduino sketch for the Soldered Inkplate 5 V2 (ESP32, 1280×720 e-ink display).

**Sketch:** `inkplate/Inkplate5V2_News.ino` — connects to WiFi, fetches the worker PNG every 4 hours via `drawPngFromWeb`, overlays the current Eastern time (with automatic DST) every minute using partial updates.

**Network helper:** `inkplate/src/Network.h/.cpp` — WiFi-only helper (connect + ensureConnected with auto-reboot).

**Board FQBN:** `Inkplate_Boards:esp32:Inkplate5V2`

**Flash:** `just flash` (default port `/dev/cu.usbserial-2144340`; override with `just flash PORT=/dev/cu.usbserial-XXXX`)
